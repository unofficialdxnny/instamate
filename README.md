 # Requires
 
 - <a href="https://www.mozilla.org/en-GB/firefox/new/">Firefox</a>

# How To Use?

- Fork the repository 
- Add your credentials in the `bot.py` file
- Save commits
- Clone your fork
- cd into the clone
- python bot.py


## How to use? (downloaded)

- Open downloaded folder
- Run the `install.bat` file
- open `bot.py` add your credentials 
- opeen `run.bat` file
